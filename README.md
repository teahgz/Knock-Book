# 📖 독서 커뮤니티 Knock Book 

## 프로젝트 소개
  - 다양한 연령층이 독서를 통해 지식을 습득하고 문화를 교류할 수 있도록 돕는 독서 커뮤니티 플랫폼입니다.
  - 다양한 기능(댓글, 대댓글, 문의사항 등)을 통해 활발한 상호작용 촉진합니다.
  - 이벤트와 독후감 공유를 통해 독서 동기 부여 및 참여 유도합니다.
